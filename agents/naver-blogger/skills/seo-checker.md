# SEO 체커 스킬

## 역할

작성된 블로그 글의 네이버 SEO 최적화 수준을 13개 항목으로 진단하고 점수를 산출합니다.

## 채점 기준 (100점 만점)

### 제목 영역 (20점)

| 항목          | 배점 | 기준                                 |
| ------------- | ---- | ------------------------------------ |
| 키워드 앞배치 | 10점 | 메인 키워드가 제목 첫 절반 안에 위치 |
| 제목 길이     | 5점  | 15~30자 사이                         |
| 클릭 유도     | 5점  | 수치, 경험 표현, 감성 수식어 포함    |

### 도입부 영역 (15점)

| 항목        | 배점 | 기준                                   |
| ----------- | ---- | -------------------------------------- |
| 키워드 포함 | 10점 | 첫 100자 내 메인 키워드 포함           |
| 공감/후킹   | 5점  | 독자의 문제 제기 또는 흥미 유발로 시작 |

### 본문 구조 영역 (30점)

| 항목          | 배점 | 기준                                                   |
| ------------- | ---- | ------------------------------------------------------ |
| 소제목 수     | 10점 | 3개 이상 (10점), 2개 (5점), 1개 이하 (0점)             |
| 소제목 키워드 | 5점  | 소제목에 서브 키워드 포함 여부                         |
| 글 길이       | 10점 | 1,500~3,000자 (10점), 1,000~1,500자 (5점), 그 외 (0점) |
| 이미지 가이드 | 5점  | 📸 마커 3개 이상                                       |

### 키워드 최적화 영역 (20점)

| 항목               | 배점 | 기준                                             |
| ------------------ | ---- | ------------------------------------------------ |
| 키워드 밀도        | 10점 | 2~3% (10점), 1~2% (7점), 3~4% (5점), 그 외 (0점) |
| 서브 키워드 분산   | 5점  | 서브 키워드가 본문에 각 2회 이상 자연스럽게 분포 |
| 키워드 스터핑 없음 | 5점  | 부자연스러운 반복이 없음                         |

### 마무리 영역 (15점)

| 항목      | 배점 | 기준                                  |
| --------- | ---- | ------------------------------------- |
| 요약 포함 | 5점  | 핵심 내용 정리 존재                   |
| CTA 포함  | 5점  | 공감, 댓글, 다음 글 유도 등 행동 유도 |
| 태그 추천 | 5점  | 해시태그 10~15개 제시                 |

## 등급 기준

```
90~100점: S등급 ⭐ — 바로 발행 가능
80~89점:  A등급 ✅ — 발행 가능 (미세 조정 권장)
70~79점:  B등급 ⚠️ — 수정 필요
60~69점:  C등급 ❌ — 상당한 수정 필요
60점 미만: D등급 🚫 — 재작성 권장
```

## 출력 형식

```
## 📊 SEO 진단 결과: [점수]/100 ([등급])

### 제목 (N/20)
- ✅/⚠️/❌ 키워드 앞배치: [점수]/10 — [설명]
- ✅/⚠️/❌ 제목 길이: [점수]/5 — [현재 N자]
- ✅/⚠️/❌ 클릭 유도: [점수]/5 — [설명]

### 도입부 (N/15)
- ✅/⚠️/❌ 키워드 포함: [점수]/10 — [설명]
- ✅/⚠️/❌ 공감/후킹: [점수]/5 — [설명]

### 본문 구조 (N/30)
- ✅/⚠️/❌ 소제목 수: [점수]/10 — [현재 N개]

- ✅/⚠️/❌ 이미지 가이드: [점수]/5 — [현재 N개]

### 키워드 최적화 (N/20)
- ✅/⚠️/❌ 키워드 밀도: [점수]/10 — [현재 N%]
- ✅/⚠️/❌ 서브 키워드: [점수]/5 — [설명]
- ✅/⚠️/❌ 스터핑 없음: [점수]/5 — [설명]

### 마무리 (N/15)
- ✅/⚠️/❌ 요약: [점수]/5 — [설명]

- ✅/⚠️/❌ 태그: [점수]/5 — [현재 N개]

### 💡 개선 제안
1. [가장 점수가 낮은 항목에 대한 구체적 개선 방법]

3. [세 번째로 낮은 항목]
```

## 자동 수정 규칙

**글 작성 워크플로우에서 호출된 경우** (SEO 진단 전용이 아닌 경우):

- 80점 미만: 문제 항목을 자동 수정하고 재채점
- 자동 수정 대상: 키워드 밀도, 소제목 키워드, 이미지 마커, 태그 수
- 자동 수정 불가 대상: 글 전체 구조, 톤앤매너 (사용자 확인 필요)
- 재채점 후에도 80점 미만이면 사용자에게 알리고 수동 개선 안내
