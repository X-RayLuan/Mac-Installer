# 경쟁 분석 스킬

## 역할

네이버 검색에서 타겟 키워드의 상위 노출 글을 분석하여 글 작성 전략을 수립합니다.

## 실행 절차

### 1단계: 상위 글 검색

웹 검색으로 네이버 블로그 상위 글을 조사합니다.

검색 쿼리:

- `site:blog.naver.com {메인 키워드}`
- `{메인 키워드} 네이버 블로그`

### 2단계: 상위 글 구조 분석

검색 결과에서 파악 가능한 정보를 분석합니다:

```
분석 항목:
- 제목 패턴: 키워드 위치, 길이, 수식어
- 글 구조: 소제목 수, 섹션 구성
- 콘텐츠 유형: 정보형/리뷰형/리스트형
- 공통 키워드: 상위 글에서 반복되는 표현
- 차별화 기회: 상위 글에서 빠져있는 정보
```

### 3단계: 전략 도출

```
## 📊 경쟁 분석 결과

### 상위 글 패턴
- 평균 제목 길이: ~자
- 공통 제목 패턴: {패턴}
- 주요 소제목 키워드: {키워드 목록}
- 공통 콘텐츠 구조: {구조}

### 차별화 전략
- 상위 글에 없는 정보: {내용}
- 개선 가능한 부분: {내용}
- 권장 앵글: {접근 방식}
```

## 주의사항

- 웹 검색 결과에서 확인 가능한 정보만 분석 (실제 글 본문 접근이 제한될 수 있음)
- 분석 결과는 참고용이며, 상위 글을 복사하거나 표절하지 않도록 주의
- 차별화 포인트를 반드시 1개 이상 제시
