# 시리즈 플래너 스킬

## 역할

대주제를 기반으로 네이버 블로그 시리즈를 기획합니다. 키워드 전략, 내부 링크 구조, 발행 순서까지 포함한 완성 기획안을 작성합니다.

## 실행 절차

### 1단계: 시리즈 범위 확정

사용자에게 확인할 사항:

- **대주제**: 시리즈가 다루는 전체 주제
- **편 수**: 원하는 편 수 (기본값: 5~10편)
- **타겟 독자**: 초보자/중급자/전문가
- **발행 주기**: 주 1회, 주 2~3회 등 (기획 참고용)

### 2단계: 하위 주제 도출

웹 검색을 활용하여 대주제에서 하위 주제를 확장합니다.

확장 방법:

```
대주제: "홈카페"

단계적 확장:
├── 입문 (장비, 원두 선택, 기본 레시피)
├── 중급 (라떼아트, 핸드드립, 다양한 추출법)
├── 심화 (로스팅, 원두 산지별 특징)
└── 응용 (시즌 메뉴, 디저트 페어링, 인테리어)
```

### 3단계: 편별 키워드 선정

각 편에 대해 간략한 키워드 리서치를 수행합니다:

- 각 편의 메인 키워드 1개
- 서브 키워드 2~3개
- 검색 의도 분류
- 글 유형 추천

### 4단계: 내부 링크 전략

시리즈 편 간의 연결 구조를 설계합니다:

```
링크 원칙:
- 이전편/다음편 순차 링크 (기본)
- 관련 주제 편끼리 교차 링크
- 1편(입문)은 전체 시리즈 목차 역할
- 마지막 편에서 1편으로 순환 링크
```

### 5단계: 발행 순서 결정

고려 사항:

- **입문 → 심화** 순서가 기본이나, SEO 관점에서 검색량 높은 편을 먼저 발행하는 것도 전략
- 시즌성이 있는 주제는 타이밍 고려
- 1편은 반드시 시리즈 소개 + 목차를 포함

### 6단계: 기획안 출력

```
## 📚 시리즈 기획: [대주제]

**타겟**: [타겟 독자]
**총 편수**: [N]편
**예상 기간**: [발행 주기 기준 N주]

---

### 시리즈 구성

| 순서 | 제목 (안) | 메인 키워드 | 글 유형 | 연결 |
|------|----------|------------|--------|------|
| 1편  | ...      | ...        | ...    | → 2편, 3편 |
| 2편  | ...      | ...        | ...    | ← 1편, → 3편 |
| ...  | ...      | ...        | ...    | ... |

### 편별 요약

**1편: [제목]**
- 키워드: [메인] + [서브 1, 서브 2]
- 핵심 내용: [2~3줄 요약]
- 글 유형: [유형]

**2편: [제목]**
...

### 내부 링크 맵

1편 ←→ 2편 ←→ 3편 ←→ ...
     ↘        ↗
       5편 (관련 주제)

---

어떤 편부터 작성을 시작할까요?
```
