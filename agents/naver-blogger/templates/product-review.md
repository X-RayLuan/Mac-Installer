# 상품 리뷰 템플릿

## 적합한 키워드

"구매", "가격", "할인", "최저가", "언박싱", "구매 후기"

## 용도

스마트스토어 셀러, 체험단, 제품 블로거를 위한 상품 중심 리뷰 템플릿.
일반 리뷰보다 구매 전환에 초점.

## 글 구조

```
[제목]: {상품명} + {경험 수식어} + {가격/구매 정보}
예: "다이슨 에어랩 멀티 스타일러 내돈내산 3개월 후기 최저가 비교"

---

[도입부] (100~200자)
- 구매 고민 과정/계기
- "결론부터 말하면 ✅추천/⚠️조건부/❌비추" (후킹)
- 구매 가격, 구매처 명시
- 메인 키워드 포함

📸 [제품 박스 또는 포장 상태 이미지]

---

[소제목 1]: 제품 기본 정보
- 정식 명칭, 모델명
- 가격 (정가 / 실구매가 / 할인 정보)
- 구매처 (쿠팡/네이버쇼핑/공홈 등)
- 주요 스펙 표

| 항목 | 내용 |
|------|------|
| 가격 | ... |
| 크기/무게 | ... |
| 핵심 스펙 | ... |
| 구성품 | ... |

📸 [구성품 전체 나열 이미지]

---

[소제목 2]: 개봉기 / 첫인상
- 패키징, 구성품 확인
- 첫 사용 느낌
- "박스를 열었을 때~", "처음 들었을 때~"

📸 [언박싱 과정 이미지 2~3장]

---

[소제목 3]: 실사용 후기 — {핵심 기능}
- 가장 중요한 기능 위주 상세 리뷰
- 일상에서 어떻게 사용하는지 시나리오
- 구체적 수치 ("배터리 10시간 → 실사용 7시간")

📸 [실제 사용 장면]

---

[소제목 4]: 장점 & 단점

**👍 장점**
1. [구체적 장점 + 근거]
2. [구체적 장점 + 근거]
3. [구체적 장점 + 근거]

**👎 단점**
1. [솔직한 단점 + 대안/해결책]
2. [솔직한 단점]

📸 [장단점 관련 이미지]

---

[소제목 5]: 구매 가이드 — 어디서 사는 게 이득?
- 플랫폼별 가격 비교
- 할인 시기/쿠폰 정보
- "제가 구매한 시점 기준~"

---

[마무리] (100~200자)
- 종합 평점: ⭐⭐⭐⭐☆ (4/5)
- 추천 대상 / 비추 대상
- "구매 고민되시면 댓글로~" CTA
- 관련 상품 리뷰 내부 링크
```

## 특징

- 가격 정보가 핵심 — 반드시 구체적 금액 포함
- "내돈내산" 강조로 광고 의심 최소화
- 구매 링크 삽입 시 "해당 링크는 제휴 마케팅~" 고지 필요
- 스펙표/비교표는 체류 시간을 크게 높임
- 시간이 지나면 가격이 변하므로 "작성 시점 기준" 명시
