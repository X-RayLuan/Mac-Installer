# 체험형 (Review) 템플릿

## 적합한 키워드

"후기", "리뷰", "사용기", "솔직", "체험", "다녀왔어요"

## 글 구조

```
[제목]: {키워드} + {기간/경험 수식어} + {솔직/리얼 수식어}
예: "에어팟 프로 2 6개월 사용 솔직 후기 장단점 총정리"

---

[도입부] (100~200자)
- 구매/방문 계기
- 사용 기간/횟수 명시 (신뢰도 향상)
- "결론부터 말하면~" 식의 후킹 (선택)
- 메인 키워드 포함

📸 [제품/장소 대표 이미지 — 직접 촬영 느낌]

---

[소제목 1]: 기본 정보 / 스펙 / 개요
- 제품: 가격, 스펙, 구매처
- 장소: 위치, 영업시간, 가격대
- 객관적 정보 먼저 제공

📸 [스펙 비교표 또는 외관 이미지]

---

[소제목 2]: 장점 — {구체적 장점 키워드}
- 3~5가지 장점을 구체적으로
- 각 장점마다 실제 사용 경험 포함
- "특히 ~할 때 진가를 발휘합니다"

📸 [장점을 보여주는 사용 장면]

---

[소제목 3]: 단점 — 솔직하게 말하면
- 2~3가지 단점 (솔직함 = 신뢰도)
- "아쉬운 점이 있다면~"
- 단점도 구체적으로 (두루뭉술하면 신뢰 하락)

📸 [단점 관련 이미지 (선택)]

---

[소제목 4]: 총평 / 추천 대상
- 별점 또는 점수 (선택)
- "~한 분들에게 추천", "~한 분들은 비추"
- 대안 제품/장소 언급 (내부 링크 기회)

---

[마무리] (100~150자)
- "N개월 써본 결론은~"
- "궁금한 점 댓글로~" CTA
- 관련 리뷰 내부 링크
```

## 특징

- 장점만 나열하면 광고처럼 보여 C-Rank 하락 → **단점 반드시 포함**
- 사용 기간이 길수록 신뢰도 높음 (1주 < 1개월 < 6개월)
- 구체적 수치가 많을수록 좋음 ("배터리 좋음" → "하루 6시간 사용 시 이틀 버팀")
- "직접 구매", "내돈내산" 같은 표현이 광고 의심을 줄임
