# 정보형 (How-to) 템플릿

## 적합한 키워드

"~하는 방법", "~하는 법", "~가이드", "~이란", "~알아보기"

## 글 구조

```
[제목]: {키워드} + {방법/가이드 수식어}
예: "아이폰 배터리 절약하는 방법 7가지 설정 꿀팁"

---

[도입부] (100~150자)
- 독자의 문제/고민 공감
- "이 글에서 ~를 단계별로 정리했습니다"
- 메인 키워드 포함

📸 [관련 대표 이미지 또는 결과물 이미지]

---

[소제목 1]: 핵심 개념/배경 설명
- 주제에 대한 기본 이해
- 왜 이것이 중요한지
- 서브 키워드 A 포함

📸 [개념 설명 이미지 또는 비교 이미지]

---

[소제목 2~N]: 단계별 방법 설명
- 각 단계를 구체적으로 설명
- 실제 수치/예시 포함
- "직접 해보니~" 경험 표현
- 서브 키워드 B, C 분산

📸 [각 단계별 스크린샷 또는 과정 이미지]

---

[소제목: 주의사항/꿀팁] (선택)
- 자주 하는 실수
- 알아두면 좋은 추가 팁
- 전문가 느낌의 인사이트

---

[마무리] (100~150자)
- 핵심 3줄 요약
- "궁금한 점은 댓글로~" CTA
- 관련 글 내부 링크
```

## 특징

- 소제목 번호 매기기 권장 (1단계, 2단계... 또는 첫 번째, 두 번째...)
- 각 단계가 독립적으로 이해 가능해야 함
- 글 하단에 전체 과정 요약 체크리스트 추가 시 체류 시간 증가
