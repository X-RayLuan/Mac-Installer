# 네이버 블로그 SEO 핵심 규칙

## 1. C-Rank (Creator Rank)

네이버가 블로거의 신뢰도를 평가하는 알고리즘.

### 글 작성 시 C-Rank 최적화 원칙

- **전문성 있는 톤 유지**: "~인 것 같아요" 대신 "~입니다", "직접 경험한 바로는~"
- **특정 주제 집중**: 한 블로그에서 잡다한 주제보다 일관된 카테고리가 유리
- **경험 기반 서술**: "직접 사용해보니", "3개월 써본 후기" 등 체험 표현 포함
- **구체적 수치/데이터 포함**: "많이 할인" 대신 "32% 할인 (정가 45,000원 → 30,600원)"

## 2. D.I.A. (Deep Intent Analysis)

네이버가 문서의 품질을 평가하는 알고리즘. 사용자의 검색 의도에 얼마나 잘 부합하는지 분석.

### D.I.A. 최적화 원칙

- **검색 의도 정확히 매칭**: "강남 맛집" 검색자는 목록+위치+가격을 원함
- **체류 시간 높이기**: 긴 글(1,500~3,000자), 이미지, 구조화된 소제목
- **포괄적 정보 제공**: 한 글에서 검색자의 질문을 완전히 해결
- **최신성 유지**: 날짜, 시즌, 최근 변경사항 반영

## 3. 키워드 최적화 규칙

### 제목 (가장 중요)

- 핵심 키워드를 제목 앞쪽에 배치
- 30자 이내 권장
- 패턴: `[키워드] + [수식어/후킹]`
- 좋은 예: "강남 맛집 추천 직장인 점심 BEST 7"
- 나쁜 예: "오늘 소개할 곳은 바로 강남에 있는 맛집입니다"

### 본문 키워드 배치

- **첫 100자**: 반드시 메인 키워드 1회 포함
- **키워드 밀도**: 전체 글 대비 2~3% (1,500자 글 기준 메인 키워드 5~7회)
- **서브 키워드**: 소제목과 본문에 자연스럽게 분산 (각 2~3회)
- **키워드 스터핑 금지**: 부자연스러운 반복은 오히려 감점

### 소제목 (H2/H3)

- 최소 3개 이상의 소제목 사용
- 각 소제목에 관련 키워드 또는 서브 키워드 포함
- 소제목만 읽어도 글의 흐름을 파악할 수 있어야 함

## 4. 검색 의도 분류

| 검색 의도 | 키워드 패턴                 | 최적 글 유형   |
| --------- | --------------------------- | -------------- |
| 정보 탐색 | ~하는 방법, ~하는 법, ~이란 | how-to         |
| 비교/선택 | 추천, BEST, TOP, ~순위      | listicle       |
| 체험 확인 | 후기, 리뷰, 사용기, 솔직    | review         |
| 비교 분석 | vs, 비교, 차이, 뭐가 나을까 | comparison     |
| 구매 결정 | 가격, 할인, 최저가, 구매    | product-review |

## 5. 패널티 회피

- **복사/표절 금지**: 다른 글 복붙 절대 불가. 네이버 저품질 직행
- **도배 금지**: 같은 키워드로 하루에 여러 글 작성 금지
- **광고성 표현 주의**: "지금 바로 구매하세요!" 과다 사용 주의
- **비공개→공개 어뷰징**: 날짜 조작용 비공개 후 공개 전환 감지됨
