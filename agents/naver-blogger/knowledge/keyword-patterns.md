# 키워드 배치 패턴

## 메인 키워드 배치 공식

### 제목 패턴

```
패턴 1: [키워드] + [수량/순위] + [부가 정보]
예: "홍대 카페 추천 TOP 5 감성 인테리어"

패턴 2: [키워드] + [경험 수식어]
예: "에어팟 프로 2 후기 6개월 사용 솔직 리뷰"

패턴 3: [키워드] + [비교 대상]
예: "갤럭시 S25 vs 아이폰 16 카메라 비교"

패턴 4: [키워드] + [방법/가이드]
예: "주식 시작하는 방법 초보자 완벽 가이드"

패턴 5: [키워드] + [시즌/시의성]
예: "2026 봄 코디 추천 직장인 출근룩"
```

### 본문 키워드 배치 맵

```
[제목] ← 메인 키워드 (앞쪽)
│
[도입부 100자] ← 메인 키워드 1회 필수
│
[소제목 1] ← 서브 키워드 A
[본문 300자] ← 메인 키워드 1회 + 서브 키워드 A 1회
[📸 이미지 1]
│
[소제목 2] ← 서브 키워드 B
[본문 300자] ← 메인 키워드 1회 + 서브 키워드 B 1회
[📸 이미지 2]
│
[소제목 3] ← 서브 키워드 C
[본문 300자] ← 메인 키워드 1회 + 서브 키워드 C 1회
[📸 이미지 3]
│
[마무리] ← 메인 키워드 1회 + CTA
[📸 이미지 4 (선택)]
│
[태그] ← 메인 + 서브 키워드 전체
```

## 롱테일 키워드 확장 패턴

시드 키워드에서 롱테일 키워드를 도출하는 공식:

```
시드: "강남 맛집"

확장 축 1 - 세분화: 강남 점심 맛집, 강남 저녁 맛집, 강남역 맛집
확장 축 2 - 상황: 강남 데이트 맛집, 강남 회식 맛집, 강남 혼밥 맛집
확장 축 3 - 음식: 강남 파스타 맛집, 강남 초밥 맛집, 강남 고기 맛집
확장 축 4 - 조건: 강남 맛집 주차, 강남 맛집 웨이팅, 강남 가성비 맛집
확장 축 5 - 후기: 강남 맛집 후기, 강남 맛집 솔직 리뷰
```

## 키워드 밀도 계산

```
키워드 밀도(%) = (키워드 출현 횟수 × 키워드 글자 수) / 전체 글자 수 × 100

목표: 2~3%

예시: "강남 맛집" (4글자), 글 길이 2,000자
→ 필요 출현: 2,000 × 0.025 / 4 ≈ 12~13회
→ 실제 권장: 자연스러움 고려하여 8~12회
```

## 해시태그 전략

```
총 태그: 10~15개
구성:
- 메인 키워드: 1개
- 서브 키워드: 3~5개
- 롱테일 키워드: 3~5개
- 시즌/트렌드 태그: 1~2개
- 지역 태그 (해당 시): 1~2개
```
