# 네이버 블로그 포맷 가이드

## 최적 글 길이

| 글 유형        | 권장 글자 수 | 소제목 수   | 이미지 수 |
| -------------- | ------------ | ----------- | --------- |
| how-to         | 2,000~3,000  | 4~6         | 5~8       |
| listicle       | 1,500~2,500  | 항목 수만큼 | 항목당 1  |
| review         | 2,000~3,000  | 3~5         | 5~10      |
| comparison     | 1,800~2,500  | 항목별 2~3  | 3~6       |
| product-review | 1,500~2,500  | 3~5         | 5~8       |

## 글 구조 공식

### 도입부 (100~200자)

- 독자의 공감/문제 제기로 시작
- 메인 키워드 자연스럽게 포함
- "이 글에서 ~를 알려드리겠습니다" 식의 예고

```
❌ "안녕하세요~ 오늘은 강남 맛집에 대해 알려드릴게요^^"
✅ "강남에서 점심 약속이 잡혔는데 어디를 가야 할지 고민이신가요?
   직장인 7년 차로 강남 일대를 샅샅이 다녀본 경험을 바탕으로,
   진짜 재방문하는 강남 맛집 7곳을 정리했습니다."
```

### 본문 (소제목 단위)

- 소제목은 명확하고 키워드 포함
- 한 섹션당 200~400자
- 구체적 정보: 가격, 위치, 시간 등 수치 포함
- 개인 경험/의견 포함 (C-Rank 가점)

### 이미지 삽입 규칙

- 300자마다 1장 (체류 시간 증가)
- 글 시작 후 첫 이미지는 2~3문장 이내
- 이미지 아래에 간단한 캡션 (키워드 포함 가능)
- 본문에서 이미지 위치를 `📸` 마커로 표시

### 마무리 (100~200자)

- 핵심 내용 1~2줄 요약
- CTA(행동 유도) 포함:
  - "도움이 되셨다면 공감/댓글 부탁드려요"
  - "궁금한 점은 댓글로 남겨주세요"
  - "다음 편에서는 ~를 다루겠습니다" (시리즈)

## 톤앤매너 가이드

### 기본 톤: 신뢰감 있는 경험자

```
✅ 권장:
- "직접 다녀와 보니~"
- "3개월간 사용해본 결과~"
- "솔직히 말씀드리면~"
- "개인적으로 추천드리는 이유는~"

❌ 지양:
- "~인 것 같습니다" (애매함)
- "~라고 합니다" (전해 들은 정보)
- 과도한 이모지 (😍🔥💯)
- "최고!", "대박!" 등 과장 표현
```

### 문체 규칙

- 존칭 사용 (~합니다, ~드립니다)
- 한 문장 40자 이내 권장
- 전문 용어 사용 시 괄호로 설명 추가
- 줄바꿈 적극 활용 (가독성)

## 인용구/강조 활용

```
네이버 블로그에서 활용 가능한 강조 요소:
- 인용구: 핵심 문장이나 팁 강조 시
- 구분선: 섹션 전환 시
- 굵은 글씨: 핵심 키워드나 중요 정보
- 리스트: 나열 정보 (가격, 메뉴, 스펙 등)
```

## 내부 링크 전략

- 시리즈 글이면 이전/다음 편 링크 삽입
- 관련 주제 글 자연스럽게 연결
- 링크는 앵커 텍스트(키워드 포함)로 삽입
- 한 글당 내부 링크 2~3개가 적정
