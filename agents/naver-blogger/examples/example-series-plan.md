> 사용자 입력: "홈카페 주제로 8편짜리 시리즈 기획해줘"

## 📚 시리즈 기획: 홈카페 완전 정복

**타겟**: 홈카페에 관심 있는 입문자~초중급자
**총 편수**: 8편
**예상 기간**: 주 2회 발행 시 4주

| 8편  | 홈카페 1년 후기 총 비용과 솔직한 장단점                | 홈카페 후기          | review     | ← 전체            |

### 편별 요약

**1편: 홈카페 입문 가이드**

- 키워드: 홈카페 입문 + 홈카페 시작, 홈카페 장비, 홈카페 비용
- 핵심 내용: 최소 장비 리스트, 5만원 세팅, 첫 커피 레시피
- 글 유형: how-to
- 역할: 시리즈 관문. 전체 시리즈 목차 포함

**2편: 그라인더 추천**

- 키워드: 홈카페 그라인더 추천 + 커피 그라인더 가성비, 수동 그라인더
- 핵심 내용: 수동/전동 그라인더 5종 비교, 가격대별 추천
- 글 유형: listicle

**3편: 원두 추천**

- 키워드: 홈카페 원두 추천 + 핸드드립 원두, 원두 구독
- 핵심 내용: 맛 프로필별 원두 7종, 구매처 비교
- 글 유형: listicle

**4편: 핸드드립 방법**

- 키워드: 핸드드립 하는 방법 + 핸드드립 레시피, 커피 내리기
- 핵심 내용: 기본/아이스/진한 레시피 3가지, 물 온도/비율 가이드
- 글 유형: how-to

**5편: 라떼 만들기**

- 키워드: 홈카페 라떼 만들기 + 우유거품, 홈카페 레시피
- 핵심 내용: 거품기 유무별 라떼 레시피, 우유 종류별 차이
- 글 유형: how-to

**6편: 커피머신 vs 핸드드립**

- 키워드: 홈카페 커피머신 추천 + 커피머신 핸드드립 비교
- 핵심 내용: 편의성/맛/비용 축으로 비교, 상황별 추천
- 글 유형: comparison

**7편: 인테리어 소품**

- 키워드: 홈카페 인테리어 + 홈카페 소품, 홈카페 꾸미기
- 핵심 내용: 가성비 소품 추천, 공간 연출 팁
- 글 유형: listicle

**8편: 1년 총 후기**

- 키워드: 홈카페 후기 + 홈카페 비용, 홈카페 장단점
- 핵심 내용: 총 투자 비용, 월 커피 지출 비교, 솔직 장단점
- 글 유형: review
- 역할: 시리즈 마무리. 전체 편 링크 포함

### 내부 링크 맵

```
1편 (입문) ← 시리즈 허브
├── → 2편 (그라인더)
├── → 3편 (원두)
├── → 4편 (핸드드립) → 5편 (라떼)
│                    → 6편 (머신 vs 드립)
├── → 7편 (인테리어)
└── → 8편 (총 후기) ← 모든 편에서 링크
```
